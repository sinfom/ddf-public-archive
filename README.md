# Dual Domain Framework Public Research Archive

The Dual Domain Framework (DDF) is an independent speculative research programme exploring whether observable physical structure can be modelled as the constrained projection of a deeper generative domain.

This repository contains public research notes, derivation chains, draft papers, dependency maps, and raw combined archive material.

The framework is developmental. Some notes are conceptual, some are formal derivation attempts, some are conditional theorems, and some are open research programmes. Claims should be read according to the status labels and context given in each note.

## Repository Structure

- `docs/` - GitHub Pages website and public entry point.
- `notes/` - organised research notes copied from the working DDF archive.
- `papers/` - public-facing paper drafts.
- `raw/` - combined source material and key indexes.
- `archive/` - historical combined files and material retained for provenance.

## Status

DDF should be read as a structural unification programme, not as a completed physical theory. It currently develops a coherent derivation chain from projection admissibility toward finite propagation, Lorentzian kinematics, Dirac/spin structure, curvature response, and an Einstein-like gravity limit, while leaving substantial open problems.
