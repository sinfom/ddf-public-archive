# Dual Domain Framework — Global Map

This document provides the structural overview of the Dual Domain Framework (DDF).  
The framework derives physical laws from projection constraints acting between an infinite generative domain Ω and the observable universe U.

## Relationship Between Foundations and Operator Notes

- Foundations (F1, F2) define the projection ontology and admissibility/spectral moderation constraints on Ω → U.  
- Operator notes (N1–N3) realise these constraints as propagation geometry and operator structure (propagation cone, Dirac operator, spin).  
- Quantum and field-theoretic notes (N4–N8) extend this operator structure into phase space and an action principle.  
- Geometric and gravity-limit notes (N9–N12) extract gauge and curvature response, culminating in a spectral-action formulation and an Einstein limit.
