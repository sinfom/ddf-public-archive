# Purpose

This note records the current progress of the Dual Domain Framework (DDF) in explaining the origin of fundamental physical constants from the projection structure.

It summarises:

- what constants have already been structurally derived
- what mathematical mechanisms produced them
- what step remains before the fine-structure constant ( \alpha ) can be derived.

------

# Core Idea of the Framework

The observable universe arises from a projection

```
Ω  →  U
```

The structure of this projection determines the physical laws and constants that appear in the observable universe.

The Dirac operator generated by the projection becomes the central mathematical object controlling:

- propagation
- quantisation
- curvature
- gauge structure.

------

# Constants Derived So Far

## 1. Speed of Light (c)

Derived from **propagation rigidity**.

The projection imposes a bound on how information propagates.

This produces a universal propagation limit:

```
propagation bound → c
```

Interpretation:

- c is the maximum transport speed allowed by the projection geometry.

Status: **Derived**

------

## 2. Planck Constant (ħ)

Derived from **phase-space quantisation constraints**.

The projection limits how precisely position and momentum can be resolved.

This produces a minimum phase-space unit:

```
phase-space resolution → ħ
```

Interpretation:

- ħ measures the smallest meaningful phase-space cell allowed by the projection.

Status: **Derived**

------

## 3. Newton’s Gravitational Constant (G)

Derived from the **spectral action of the Dirac operator**.

Steps:

```
Dirac operator
→ curvature appears in D²
→ heat-kernel expansion
→ Einstein–Hilbert action
→ gravitational coupling
```

This produces the gravitational constant:

```
G ∼ ħ c / Λ²
```

where Λ is a spectral cutoff scale imposed by the projection.

Interpretation:

- G measures how strongly curvature responds to energy.

Status: **Derived structurally**
Exact value depends on the projection cutoff scale Λ.

------

# Spectral Cutoff from Projection

The projection restricts admissible states in phase space.

This implies a bounded Dirac spectrum:

```
|λ| ≤ Λ_proj
```

This produces a natural spectral cutoff

```
Λ = Λ_proj
```

which determines the gravitational coupling.

Status: **Derived existence of cutoff**
Exact numerical value still unknown.

# Next Target Constant

The next constant is the **fine-structure constant**

```
α ≈ 1/137
```

which measures the strength of electromagnetism.

------

# Why α Appears Later

In the spectral expansion of the Dirac operator:

```
a₂ term → curvature → gravity
a₄ term → gauge fields → electromagnetism
```

Gravity appears earlier because it depends only on geometry.

Electromagnetism appears later because it depends on the structure of charged particles.

------

# The Remaining Gap

To compute α the framework must determine:

```
what particles exist
what charges they carry
```

These properties determine the strength of the electromagnetic interaction.

The missing step is therefore:

```
projection → particle spectrum and charges
```

Once the projection uniquely determines particle charges, the value of α can in principle be derived.
