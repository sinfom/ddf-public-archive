# Operator Chain

The operator layer develops the route from admissibility constraints to propagation structure and first-order operator form.

Core themes include:

- propagation rigidity
- hyperbolic cone structure
- wave operator emergence
- Dirac factorisation
- spin structure
- gauge and fibre algebra selection
- curvature and spectral-action interfaces

The full operator notes are in `notes/02_operator_notes/` in the repository.
